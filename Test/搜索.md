##搜索##

1. 方格型搜索
    * 方位表示
      ```cpp
        int dx[8]={-1,-1,0,1,1...}          
        int dy[8]={0,1,1,1,0...}
        for(int i=0;i<8;i++){
            x+dx[i],y+dy[i]
        }
      ```
    * 占据：
      * 对角线表示（九皇后）
      ```cpp
        bool c1[maxn];
        bool cu[maxn * 6], cd[maxn * 6]; 
        !c1[i] && !cu[i - row + n] && !cd[i + row]
      ```  
      * 外圈加一层技巧（luogu p1162）
      * 传送门 
        * 不要试图写双向bfs。。。。
        * 必须使用传送门型（陷阱型）洛谷P1825
          （1）
            ```
              3 6

              ######

            ``` 
            正确输出应该是4
          （2）
            ```
              3 6
            ``` 

    * 双向bfs
      * _真的很厉害_ 18ms过P1032 双向bfs可以减半搜索树的深度！！！！！10层减少为5层减少了太多空间和时间，尤其是处理一个节点是时间常数很大时要双向优化，题目特征一般也很明显，n>20多了，而且给出了初态和终态
